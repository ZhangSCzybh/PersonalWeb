## 1. 产品概述

### 1.1 版本简介

PersonalWeb V2.0 在 V1.0 基础上新增**衣橱管理模块**，为用户提供智能化的服饰管理和 AI 穿搭生成服务。

### 1.2 V2.0 新增功能

| 功能 | 描述 |
|------|------|
| 衣橱管理 | 服饰上传、分类管理、多维度筛选 |
| AI 穿搭 | 自定义搭配生图 + 随机穿搭推荐 |
| 图片处理 | 自动抠图去背景 |

### 1.3 与 V1.0 关系

V2.0 为增量更新，不影响 V1.0 现有功能

## 2. 功能需求

### 2.1 衣橱页面 (/wardrobe)

| 功能点 | 描述 | 优先级 | 详细规格 |
|--------|------|--------|----------|
| 衣橱分类管理 | 默认分类，可修改添加 | P0 | 5大分类：外套、内搭、下装、鞋子、包包 |
| 服饰上传 | 图片上传/拍照，支持抠图去背景 | P0 | 移动端拍照上传，自动背景抠图 |
| 颜色选择 | 色系选择 | P0 | 黑/白/灰/红/黄/绿/褐/紫/蓝色系 |
| 喜好值 | 1-5级评分 | P0 | 必选项，影响AI穿搭推荐 |
| 季节选择 | 春夏秋冬多选 | P0 | 必选项，用于季节筛选 |
| 可选属性 | 购物时间、品牌、面料、尺码、标签、备注、价格 | P1 | 非必填扩展信息 |
| 衣橱筛选 | 多维度筛选服饰 | P0 | 按分类、颜色、季节、喜好值筛选 |
| AI穿搭生成 | 选择服饰+关键字生图 | P1 | 结合衣橱服饰和AI生成穿搭 |
| 随机穿搭 | AI随机搭配推荐 | P1 | 基于衣橱现有服饰生成穿搭方案 |
